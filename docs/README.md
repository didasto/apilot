# Apilot Documentation

> Build production-ready REST APIs in Laravel with minimal boilerplate.

Apilot is a Laravel package that turns Eloquent models and custom services into fully-featured REST APIs in minutes. Define which fields are filterable, sortable, and paginated — Apilot handles the routing, validation wiring, pagination metadata, and OpenAPI documentation automatically, while a lifecycle hook system gives you full control without having to rewrite core methods.
